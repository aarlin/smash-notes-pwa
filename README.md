# Smash Notes - Ionic PWA

Create Super Smash Bros. Ultimate match up notes 

Installable PWA  

Firebase + Angular + Ionic  

## Install

`npm install`  
`ionic serve -l`  



# Setup Firebase

Edit .env.sample to .env  
Plug in your firebase configuration for each of the environment variables
